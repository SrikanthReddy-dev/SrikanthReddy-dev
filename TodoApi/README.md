# TodoApi

A simple .NET 6 Minimal API example for managing todo tasks. This project demonstrates basic CRUD endpoints and Swagger integration.

## Prerequisites
- [.NET 6 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)

## Running the API

```bash
# Restore dependencies
# dotnet restore

# Run the API
# dotnet run
```

By default the API runs on `https://localhost:5001` with Swagger UI available at `/swagger`.
